1 DESCRIPCION DEL PRODUCTO:
El sistema consiste en un programa que permite a los usuarios crear y gestionar una lista de compra.
El usuario establece un presupuesto limite (400 mil pesos), el cual se va reduciendo automaticamente a medida que se agrega productos.

2 OBJETIVOS
Facilitar la organizacion de compra mediante:
- Control de presupuesto
- Organizacion de productos
- Evitar duplicados
- Clasificacion eficientes

3 FUNCIONALIDADES PRINCIPALES 
1. Lista
- Agregar productos a la lista
- Ingresar precio de cada producto
- Finalizar la lista con una opcion de fin

2. PRESUPUESTO
- Definir un limite de gasto
- Restar automaticamente el valor de cada producto
- Mostrar alerta cuando se alcance o supere el limite

3. DATOS
- Mostrar lista ordenada alfabeticamente
- Agrupar productos por categoria (frutas, lacteos, verduras).

4. VALIDACION 
- Contar la cantidad total de productos
- Verificar que no haya productos repetidos

4 SERVICIO DEL SISTEMA
- Recordatorio de compra
- Organizacion automaticas de datos
- Aviso de limite del presupuesto

6. REQUISITOS FUNCIONALES
- El sistema debe permitir ingresar productos y precios
- Debe calcular el total automáticamente
- Debe impedir o alertar productos duplicados
- Debe ordenar los productos
- Debe agrupar por categorías

7. REQUISITOS NO FUNCIONALES
- Interfaz simple y fácil de usar
- Respuesta rápida al ingresar datos
- Claridad en la visualización de resultados

8. CRITERIO DE EXITO
- El usuario puede completar una lista sin errores
- El presupuesto se calcula correctamente
- La lista final está organizada y sin duplicados
